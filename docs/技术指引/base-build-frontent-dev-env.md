===

---

解压build目录下的`ndoe_moduels.rar`到build目录，形成`build/node_modules/gulp`这些路径就对了

然后cmd到build目录下执行`gulp`命令即可开始开发

- 前端目录说明

	`frontent/src-home`：这是前台的源代码目录，里面都是HTML、CSS、JS什么的

	`frontent/mock-home`：这是前台的模拟数据目录，模拟数据定义教程请见[返回模拟数据](http://www.kkh86.com/it/gulp-build-dev-env/guide-mock-response-data.html)
	
	后台没开始搭暂时不介绍了
